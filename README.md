## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The development server will start at `http://localhost:3000`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Brand Information

- **Company Name**: ELLOWRING SOFTWARE SOLUTIONS
- **Logo**: Yellow ring with white center dot (3D animated)
- **Color Theme**: 
  - Primary: Yellow (#FFD700)
  - Secondary: White (#FFFFFF)
  - Background: Black (#000000)

## Pages

- **Home** - Hero section with 3D logo, services preview, and company overview
- **About** - Mission, vision, values, and company statistics
- **Services** - Detailed service offerings
- **Team** - Team member profiles with photos and social links
- **Blog** - Blog post listings
- **Contact** - Contact form and company information

## Customization

All placeholder content can be easily replaced in the respective page components. Update:
- Company information in Footer and Contact pages
- Service descriptions in Services page
- Team member information in Team page
- Blog posts in Blog page
- Logo image (currently using 3D generated logo)
